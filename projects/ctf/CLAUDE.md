# Project: CTF Puzzle-Solving Agent

## Context
This project builds an AI agent that autonomously solves legal CTF challenges.

## Agents
The available agents are defined in .claude/agents/*.md

## Workflow
Always follow the following workflow:
- Per challenge, start challenge manager agent.
- Before terminating, check whether you have followed this workflow.

## Tech stack
- python 3

## Commands
- `make init` sets up needed packages and resources
