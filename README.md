# Claude Agent Projects

A collection of AI agent projects built with Claude Code.

## Structure

Each project lives in `projects/<name>/` with its own `CLAUDE.md`, dependencies, and agents.

## Projects

- [`projects/ctf/`](projects/ctf/) — Autonomous CTF puzzle-solving agent

## Agents

Shared agents are defined in each project's `.claude/agents/*.md`. Projects may define their own agents in their subdirectory.
