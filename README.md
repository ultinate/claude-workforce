# Claude Agent Projects

A collection of AI agent projects built with Claude Code.

After working with Claude for a short time, I came to realize that **this repository is now superseded** by just the following prompt:

*I want to develop a [...] to achieve [...] using [...]. Start by rephrasing the task in your own words, flag any ambiguities, and then present me with 2–3 options for how to structure a team of Claude agents, including their roles and communication flow.*

## Structure

Each project lives in `projects/<name>/` with its own `CLAUDE.md`, dependencies, and agents.

## Projects

- [`projects/ctf/`](projects/ctf/) — Autonomous CTF puzzle-solving agent
- [`projects/arduino/`](projects/arduino/) — Develop a heating controller using Arduino
